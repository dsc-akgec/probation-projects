Sentiment analysis on tweets
The sole purpose of this project is to fetch public tweets based on the asked keyword,analyse it and classify  percent of the positive,negative or neutral  
The module used are:

tweepy <pip install tweepy> 
In order to fetch tweets through Twitter API, one needs to register an App through their twitter account,and its access in python is provide through tweepy.

textblob <pip install textblob>
A python module build on top of nltk module,providing common natural language processing(NLP)tasks such as tokenization,noun phrase extraction,sentiment analysis and more

matplotlib <pip install matplotlib>
